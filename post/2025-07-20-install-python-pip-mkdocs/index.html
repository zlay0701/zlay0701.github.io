安装绿色版python

下载绿色安装包Windows embeddable package (64-bit)

https://www.python.org/ftp/python/3.13.5/python-3.13.5-embed-amd64.zip

没有git则安装git https://git-scm.com/

添加环境变量

win11->系统->系统信息->高级系统设置->环境变量

系统变量里找到path->编辑->编辑文本

在末尾添加D:\dev_software\python-3.13.5-embed-amd64

检查是否安装成功

在任意目录右键-> open git base here

正确输出版本号则安装成功
